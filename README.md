# Country Data Map
By: Douglas Leidgen

# Stack
Angular CLI: 18.2.5
Node: 22.7.0
Package Manager: npm 10.8.2
OS: darwin arm64
Angular: 18.2.5

# Description
Angular app that displays and svg world projection map.  When the mouse pointer is placed over a country an API call is made to the World Bank site and statiscal data is displayed in a text box.
The UI also has functionality that allows a country code to be entered to retrieve data from the API.
